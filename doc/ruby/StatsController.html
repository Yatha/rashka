- (String) evacuate

Note:

POST stats/evacuate через AJAX

Обрабатывает нажатие по ВАЛИТЬ. Увеличивает статистику и готовит случайную причину для формы “рассказать друзьям”

Examples:

$.ajax "../stats/evacuate",
  type: "POST"
  dataType: "json"

def evacuate
  @stats.increase_people_left
  reason = Reason.random.text

  respond_to do |format|
    format.json { render :json => reason.to_json }
  end
end

def update_reason

  reason.try_to_save
end
